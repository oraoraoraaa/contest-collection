# Contest Collection

Collect good computer science contest problems and list useful skills gained from solving various contest problems.

# How to Use

## Problems Themselves

The practice problems themselves are directly as files in this repository.

The name of every folder in the root demonstrates the source of the practice problems inside the folder.

- [Baidu-Pinecone](https://github.com/oraoraoraaa/contest-collection/blob/main/Baidu-Pinecone): The problems are from a special online class (Baidu Pinecone Class) of Xi'an Jiaotong University.

## Skills and Other Information

About some useful skills and other information gained from solving numerous contest problems, move to the [wiki](https://github.com/oraoraoraaa/contest-collection/wiki) page.
